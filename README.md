# Morpheus AI Platform

Build production-ready AI agents in minutes, not weeks.

## Quick Start

```bash
npm install
cp .env.example .env
npm run dev
```

## Build Modes

- **Mock**: Instant builds, no API key needed — perfect for demos
- **Real**: Uses `.env` Gemini key (free tier, rate limits apply)
- **BYOK**: Bring your own paid key — no rate limits

## Demo

1. Visit http://localhost:5173
2. Click "Try Live Build"
3. Watch an agent build in 2-3 seconds

## Technology

- React 19 + TypeScript
- Gemini 3 Extended Thinking
- Multi-mode architecture (Mock / Real / BYOK)
